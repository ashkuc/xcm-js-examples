# xcm-js-tools examples

The runnable examples for [XCM JS tools](https://github.com/open-xcm-tools/xcm-js-tools).

## Usage

### Install Yarn

```bash
npm install -g yarn
```

### Install dependencies

```bash
yarn install
```

### Run examples

```bash
yarn transfer-fun
yarn transfer-non-fun
yarn sorting-assets
yarn sanitize
yarn estimate
yarn convert
```
